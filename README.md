# HAU CVs

<img src="https://github.com/allanchua101/vue-cvs/blob/main/imgs/banner.png" alt="MarineGEO circle logo" style="max-width: 720px; width: 100%; margin-left: auto; margin-right: auto; display: block;"/>

HAU CVs is a VueJS-based web application used for presenting student resumes in a single page architecture setup.


### Technology Stack

- VueJS for template rendering
- Vuetify for UI framework
- Vue Router for SPA routing capabilities.
- JavaScript as core language

### Pre-requisites

List of software you need to run this application:

- NVM
- NodeJS v12.18.3
- VUE CLI installed globally
- Shell application
